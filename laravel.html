  • Install Composer 
    https://getcomposer.org/Composer-Setup.exe

  • Laravel Package Installation 
    composer global require laravel/installer
Run This Command In CMD To Install Laravel Package
  • Create Laravel Project 
    laravel new Project-Name
Run This Command In CMD Where you Want To Install Laravel Project
  • Environment Configuration 
    Open Env File Present In root Directory And Enter Your Details 

APP_NAME        =   Project-Name
APP_URL         =   Project-Url

DB_CONNECTION   =   Your Database Connection
DB_HOST         =   Your Database Host
DB_PORT         =   Your Database Port
DB_DATABASE     =   Your Database Name
DB_USERNAME     =   Your Database Username
DB_PASSWORD     =   Your Database Password
  • Echo In Laravel 
    {{ $Variable-Name }}

    <div>
    @foreach ($users as $user)
